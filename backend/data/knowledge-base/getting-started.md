# Acme Cloud — Getting Started

## Creating your first project
After signing up, click "New Project" on the dashboard. Give it a name and choose
a region. Projects in the EU region store all data within the EU. The region
cannot be changed after creation, so choose carefully.

## Inviting your team
Go to Settings → Members → Invite. Enter teammates' email addresses and assign a
role: Owner, Admin, Editor, or Viewer. Invites expire after 7 days. There is no
limit on the number of Viewer seats on any paid plan.

## API keys
Generate API keys under Settings → Developer → API Keys. Treat keys like
passwords. Keys can be scoped to read-only or read-write. If a key is leaked,
revoke it immediately — revocation takes effect within a few seconds.

## Supported integrations
Acme Cloud integrates with Slack for notifications, GitHub for deploy triggers,
and webhooks for custom workflows. Zapier support is on the roadmap but not yet
available.
